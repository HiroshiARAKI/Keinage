# Keinage 商標ポリシー

Keinage の名称、ロゴ、および公式ブランド表現は、Hiroshi Araki が管理する商標またはブランド資産です。

Apache License 2.0 はソースコードを利用する権利を許諾しますが、公式な承認や提携を示唆する形で Keinage の名称、ロゴ、公式ブランド表現を使用する許可は含みません。

許可される利用:

- プロジェクトを「Keinage」と呼ぶこと
- 「powered by Keinage」または「based on Keinage」と表記すること
- ドキュメント、記事、社内向けデプロイ資料で名称を使用すること

許可なく利用できない例:

- 「Keinage Cloud」という名称で公開ホスティングサービスを運営すること
- 第三者のホスティングサービスの主要ブランドとして Keinage ロゴを使用すること
- Keinage の公式サービスであると主張すること
- 混同を招く類似ドメイン、サービス名、ロゴを使用すること
